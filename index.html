Gaming Passion Project App


I created a dynamic website that celebrates the fun of PC gaming, with a special focus on Skyrim, one of the most iconic open-world games. As a front-end developer, I used HTML, CSS, and JavaScript to bring the site to life. The homepage features a visually engaging layout that captures the adventure of Skyrim, complete with links to download Steam and the Skyrim game itself. To enhance the user experience, I integrated a dark mode toggle button, allowing visitors to switch between light and dark themes with a smooth JavaScript function.





Weather App

I developed a dynamic weather app/website that provides real-time weather updates for any city around the world. As a front-end developer, I leveraged advanced JavaScript functions and CSS to create an interactive and visually appealing platform. The app displays live weather conditions, including temperature, humidity, and a detailed, up-to-date date and time for the current day. It also features a comprehensive weekly forecast, complete with weather icons that visually represent real-time conditions. This project highlights my expertise in both JavaScript for data handling and CSS for creating an intuitive, engaging user experience.
